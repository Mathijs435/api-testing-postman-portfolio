# 🚀 API Testing Portfolio: Restful-Booker (Postman)

## 🇩🇪 Projektbeschreibung
Dieses Repository enthält meine API-Testsuite für die "Restful-Booker" Plattform. Ich habe dieses Projekt erstellt, um meine Fähigkeiten im Bereich API-Testing und Testautomatisierung im Rahmen meiner ISTQB-Vorbereitung zu demonstrieren.

### Kernpunkte des Projekts:
- **Test-Tool:** Postman (Desktop-App)
- **Validierung:** Automatisierte JavaScript-Assertions zur Überprüfung von Statuscodes (200 OK / 404 Not Found) und Datentypen.
- **Strategie:** Anwendung von Positive- und Negative-Testing-Szenarien.
- **Dokumentation:** Detaillierte Erläuterungen und Screenshots sind in meinem Notion-Portfolio hinterlegt.

---

## 🇺🇸 Project Overview
A collection of API tests for the "Restful-Booker" API. This project showcases my ability to perform manual and automated API testing using Postman.

### Features:
- **Automated Assertions:** Scripts to verify response integrity.
- **Status Code Validation:** Ensuring correct server responses for various endpoints.
- **JSON Handling:** Practical experience with JSON data structures.

---

## 🛠 How to Import and Run
1. Download the `.json` file from this repository.
2. Open **Postman**.
3. Click on **Import** and select the downloaded file.
4. You can now view and run all test cases with their automated scripts.
